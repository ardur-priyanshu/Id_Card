# Social Distancing ID Card 📛

A wearable ID card that alerts when someone comes 
too close, built using an ultrasonic sensor.

## 🎯 Objective
Build a wearable safety device during COVID times that 
alerts the user when social distancing is violated.

## 🛠️ Components Used
- Ultrasonic Sensor (HC-SR04)
- Arduino Nano
- Buzzer
- LED
- Small LiPo Battery
- Custom casing fit for ID card size

## ⚙️ How It Works
- Ultrasonic sensor continuously measures distance
- If someone comes within set threshold distance
- Buzzer and LED trigger immediately as alert
- Compact enough to wear like a regular ID card
